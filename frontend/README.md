# Prompt Defense Frontend

Electron + React + TypeScript browser shell for Prompt Defense.

## Scripts

- `npm run dev` starts Vite with renderer hot reload.
- `npm run electron:dev` starts Vite, watches Electron TypeScript, and relaunches Electron on main-process changes.
- `npm run build` type-checks and builds Electron plus renderer assets.
- `npm run lint` runs ESLint.
